## ComposedQL

**!!! UNDER DEVELOPMENT !!!**

[![NPM][npm-image]][npm-url] [![Build Status][travis-image]][travis-url]

ComposedQL is a query language that aims URI friendly queries.


This module provides composed query parser and specifications.

### Installation

```
npm install composedql
```

### Usage

```javascript
var cql = require('composedql');
```

#### parse

Parses given composed query

```javascript
var query = cql.parse('first_name,last_name,address.city,settings.foo.bar');

// [
//   {
//     "type": "field",
//     "name": "first_name"
//   },
//   {
//     "type": "field",
//     "name": "last_name"
//   },
//   {
//     "type": "field",
//     "name": "address",
//     "properties": [
//       {
//         "type": "field",
//         "name": "city"
//       }
//     ]
//   },
//   {
//     "type": "field",
//     "name": "settings",
//     "properties": [
//       {
//         "type": "field",
//         "name": "foo",
//         "properties": [
//           {
//             "type": "field",
//             "name": "bar",
//             "properties": [
//               {
//                 "type": "field",
//                 "name": "qux"
//               }
//             ]
//           }
//         ]
//       }
//     ]
//   }
// ]
```
